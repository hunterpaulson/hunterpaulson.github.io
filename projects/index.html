typeGPT.dev

Can you type faster than GPT-2?

runs GPT-2 locally in-browser using WebGPT and generates typing tests using only words in the selected wordset.

I built this to learn local llm inference and also so I could practice typing probable word sequences instead of words sampled uniformly at random.
