For this site I still prefer reusable HTML and CSS over manually typing every box-drawing character. ASCII is great for one-off sketches, but HTML blocks are easier to compose, theme, and reuse across essays.

This direction feels strongest here: reusable HTML blocks for structure, occasional ASCII for charts, and SVG only when connectors need more precision than CSS should be forced to provide.
